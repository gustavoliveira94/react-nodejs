# Project - React And NodeJS


## Installation

```sh
cd lyncas-teste
npm install or yarn install
```

## How to execute

```sh
cd api/server
node server
```
```sh
npm start
```

## Built With

* [REACT JS](https://reactjs.org/) - The web framework used
* [MONGOOSE](https://www.npmjs.com/package/mongoose) - Package NPM
* [REDUX](https://redux.js.org/) - Lib
